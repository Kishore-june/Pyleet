# PyLeet - LeetCode Easy Platform

A premium learning platform for mastering LeetCode Easy problems using Python.

## Features

- **30+ Curated Problems**: The most asked Easy problems in interviews.
- **Deep Explanations**: Step-by-step breakdown of solutions.
- **Company-Wise Filters**: See what Google, Amazon, and Microsoft are asking.
- **AI Tutor**: Integrated AI to help you understand complex concepts.
- **Progress Tracking**: Save your solved problems locally.

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- React
- Vite
- Vanilla CSS
- Python (Solution logic)

## Deployment

This project is ready to be deployed to GitHub Pages.

1. Build the project:
   ```bash
   npm run build
   ```
2. The output will be in the `dist` folder.
